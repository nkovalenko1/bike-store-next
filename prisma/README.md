# Prisma Database Setup

## Первоначальная настройка

1. **Настройте переменные окружения**
   ```bash
   cp .env.example .env
   # Отредактируйте .env и укажите DATABASE_URL
   ```

2. **Создайте миграцию**
   ```bash
   npm run db:migrate
   ```
   Это создаст миграцию и применит её к базе данных.

3. **Заполните базу данных начальными данными**
   ```bash
   npm run db:seed
   ```

## Доступные команды

- `npm run db:generate` - Генерирует Prisma Client
- `npm run db:push` - Применяет изменения схемы без миграции (для разработки)
- `npm run db:migrate` - Создаёт и применяет миграцию
- `npm run db:studio` - Открывает Prisma Studio для просмотра данных
- `npm run db:seed` - Заполняет базу данных начальными данными
